# Toolbox Scripts

Coleção de scripts para inicializar projetos com stacks pré-configuradas.

## Como usar

```bash
bash <pasta>/setup_xxx.sh nome_do_projeto
```

Cada pasta contém o script de setup e um README com os detalhes da stack.

## Scripts disponíveis

| Script | Tecnologia | Descrição |
|---|---|---|
| [Setup Django](django/README_setup_django.md) | Python / Django | Inicializa projeto Django com uv, HTMX, Alpine.js, Tailwind + DaisyUI e autenticação |
